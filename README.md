# Expense Tracker Dashboard

A modern and responsive Expense Tracker Dashboard built using React, Tailwind CSS, and Recharts. The application helps users track, manage, and visualize their personal expenses through interactive charts, categorized transactions, and a clean dashboard interface.

## Features

* Dashboard overview with expense statistics
* Monthly spending distribution chart
* Recent expenses section
* Expense management table
* Category-based expense badges
* Search and filter functionality
* Custom dropdown components
* Add Expense modal popup
* Responsive and modern UI
* Smooth hover and transition effects

## Tech Stack

* React.js
* Tailwind CSS
* Recharts
* Lucide React Icons
* Vite

## Highlights

* Component-based scalable architecture
* Reusable UI components
* Modern SaaS-style dashboard design
* Responsive layout for different screen sizes
* Clean and maintainable folder structure

## Project Goal

The goal of this project is to showcase frontend development skills by building a professional dashboard interface with reusable components, responsive layouts, state management, and modern UI practices using React and Tailwind CSS.
